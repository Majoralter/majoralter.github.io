# The thing about resolutions
It's that time again when people (like me) outline all the things that they'll do to make the new year a memorable and/or productive one. But what usually happens is that they forget all about those resolutions halfway through January and haphazardly let the year slowly (really quickly if you're above 20) pass by. Now the problem isn't setting goals or having resolutions, it's how those goals or resolutions are set.

# Yet another year
I've been setting goals at the start of each year since 2022, but I usually let them die in my notes app never to be seen again (until the next year). I want this year to be different (don't roll your eyes), seriously I really do. So the system I'm going to follow is to set goals for each quarter that will carry-over into the next quarter. I'm going to set goals for each month at the start of that month (that means I'm going to regularly update this post).


## January
My goals for this month:
* I've been working as a solo frontend dev on a client project since September and I want to finally be free from this project. Preferrably in the first couple of weeks of this month.
* I want to start my project/startup that I've been planning with a friend since August-ish of last year.
* I'm currently reading 3 books (weird, I know) and I want to finish them. The books are The lost world, Howl's moving castle and Mistborn: The final empire.
* I really want to take more pictures this year (my potato phone makes me forget that the camera app exists) so I might as well start now. 

And that's it for January :) See you in February.
